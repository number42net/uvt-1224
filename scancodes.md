# Introduction
The below HEX values represent the binary code sent by each keypress. On release, the scan code + 1 is sent instead. 
Example:
| Key             | Code on press | Code on release |
| ----------------|---------------|-----------------|
| Hold Screen     | 2E            | 2F              |
